# Яндекс Маркет — юнит-экономика

Streamlit-приложение для расчета юнит-экономики Яндекс Маркета по моделям FBS / FBY.

## Что изменено
- простой русский шаблон товаров;
- автоматическое определение категории по названию товара;
- расчет комиссии и логистики без отдельного файла настроек;
- выбор налоговой системы в интерфейсе;
- мгновенный пересчет рекомендованной цены при изменении параметров;
- выгрузка результата в Excel.

## Запуск
```bash
pip install -r requirements.txt
streamlit run app.py
```
